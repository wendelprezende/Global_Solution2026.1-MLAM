Tema do Projeto: Análise Estatística da Economia Espacial utilizando dados reais de missões espaciais.
Objetivo: Este projeto tem como objetivo aplicar conceitos de Estatística Descritiva, Distribuição de Frequências e Visualização de Dados por meio da linguagem Python, utilizando uma base real de missões espaciais.

A análise busca identificar padrões relacionados à participação das organizações espaciais, custos das missões e evolução do setor aeroespacial, contribuindo para a compreensão dos desafios da nova economia espacial.

Base de Dados: Space Missions (1957–2022)
Fonte: Kaggle

A base contém informações sobre:
Empresas responsáveis pelos lançamentos;
Missões espaciais;
Datas dos lançamentos;
Custos das missões;
Status das missões;
Locais de lançamento.

Tecnologias Utilizadas
Python
Pandas
Matplotlib
NumPy

Resultados Obtidos

Foram realizadas:
Tabelas de distribuição de frequências;
Gráficos estatísticos;
Análises univariadas;
Cálculo de medidas de tendência central;
Cálculo de medidas de dispersão;
Cálculo de quartis.
Os resultados demonstraram a concentração histórica de lançamentos em poucas organizações e a crescente participação da iniciativa privada no setor espacial, além da grande variabilidade dos custos das missões espaciais.

Conclusão
A utilização de técnicas estatísticas permitiu transformar dados históricos do setor aeroespacial em informações relevantes para apoio à tomada de decisão e compreensão dos desafios da nova economia espacial.
